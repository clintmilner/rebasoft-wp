MAC Auditor - Installation

Installation is achieved by executing the MAC Auditor installation wizard. This wizard guides administrators through the installation or updating process and allows the specification of a number of parameters – it should only take a few minutes to install the system. The latest version can be downloaded from here »

The installer can be run whether performing a new installation or updating an existing system. In the case of an upgrade, data is preserved; Rebasoft does, however, advise administrators to perform a data back-up regularly and especially prior to system upgrade.

To start installation (or upgrade), simply execute the installer:


  1. Click next...


  2. Click “I Agree” to accept the licence agreement...


  3. Select components to install and click next..

    4. Note:
    If MAC Auditor determines an incorrect JRE Version (or JRE is missing), a message will be displayed re-directing to download the approiate JRE and the installation wizard terminated....

  4. For threads setting please see system parameters for more details.

    5. Note:
    If the http or https ports entered are already in use by another application, the installation will complete, but an entry stating “bind failed, port already in use” will appear in the MAC Auditor log, and the MAC Auditor service will fail to start.

  5. Specify drive and directory for installation click next....

    Note:
    The Java installer does not permit spaces in the specification of the destination folder


  6. MAC Auditor installs as a system service, a command window should appear during this process


  7. Once MAC Auditor installation is complete (administrators can view the details of where files have been installed by clicking the details button) click next...


  8. The installation is complete
