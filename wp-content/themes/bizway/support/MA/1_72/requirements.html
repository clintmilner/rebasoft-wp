MAC Auditor - requirements

MAC Auditor can run on Windows or LINUX platforms:

Hardware requirements

  • Intel Xeon or AMD similar processor.
  • 2Gb RAM, although performance will benefit from increased RAM, as data is cached in memory prior to writing to the database.
  • 80 Gb SATA 7200rpm+ disk (or greater depending on network size and period of storage required).

Software requirements

  • Windows 2003 server or later
    or
  • LINUX
  • Java Run time (JRE) V1.6 or later installed

MAC Auditor is supported in a Virtual PC environment (eg: VMWare)

Browser requirements

  • Internet Explorer V6 or later, Mozilla V3 or later (Javascript must enabled in your browser)
  • 1200 x 1024 pixel screen resolution is recommended
