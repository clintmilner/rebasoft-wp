Installing MAC Auditor

Installation is achieved by executing the MAC Auditor installation wizard. This wizard guides administrators through the installation or updating process and allows the specification of a number of parameters – it should only take a few minutes to install the system.

The installer can be run whether performing a new installation or updating an existing system. In the case of an upgrade, data is preserved; Rebasoft does, however, advise administrators to perform a data back-up regularly and especially prior to system upgrade.
