New features for V2

MAC Auditor V2 sees the introduction of new features built upon the previous release and based upon customer requests and requirements:

  • Active test
    active test allows the administrator to set up, on a per subnet basis, a connectivity test base upon a port number - this allows an operator to determine whether an IP address is currently active on the network and which ports are available on that IP. Quick service check - cross check with NetFlow
  • White list
    allows an administrator to watch a site and report on any newly discovered MAC addresses, and alert on it.
  • Sites:
    • Nesting of sites
      allows a structure of sites and devices to be set-up to match your network topography
    • Visuals and maps related to sites
      allows administrators to set a visual view for each site and arrange child sites and devices on that visual - as an aid to help MAC Auditor users to quickly assimilate physical location information about the site or sites.
  • PostgreSQL database

To support these new features, there have been a number of minor changes to the application which are described in the sections below. Please see V2.0 release notes
