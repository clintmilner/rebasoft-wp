New features for V2.1

Rebasoft Auditor V2 sees the introduction of new features built upon the previous release and based upon customer requests and requirements:

  • Tagging - the ability to index and report on information imported and associated in Auditor
  • The ability to store and search on DNS cached results for the network
  • User lookup for Windows Active Directory on from Auditor on Linux
  • Various performance and small usability enhancements.

To support these new features, there have been a number of minor changes to the application which are described in the sections below. A list of changes can be viewed in Rebasoft Auditor - V2.1 Release notes
