Introduction

Application Auditor is designed to use multiple sources of data from your network infrastructure to report on application usage. With application usage information organisations can:

• understand and enforce acceptable use policies

• troubleshoot performance issues

• improve perimeter security

• identify potential zero day threats

• and more

Application Auditor provides the ability to both generate summary executive reports as well as facilitating deep, dive forensic analysis down to individual network conversations.

Application Auditor, as a single environment provides a future proof, multi-platform coverage system. This saves organisations:

• money in procuring and maintaining multiple systems

• time and effort in deploying and learning multiple, disparate toolsets

Closely coupled with Rebasoft’s Auditor Core, it is now possible to obtain true end-to-end view:

1) Who (user awareness)

2) What (physical device type)

3) Where (location awareness)

4) Which application has been accessed

5) When, over time
