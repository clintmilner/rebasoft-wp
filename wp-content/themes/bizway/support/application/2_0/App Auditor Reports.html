Reports

Once Application Auditor starts receiving data from source devices, it may take a few minutes for report data to appear. Each flow datagram received is automatically processed and stored in both the summarised reporting database and forensic database (although this is the same database instance). Data management in the two databases is handled separately, and operates independently.

After 5 minutes of data processing, reports should be visible.

Overview Reports

A dashboard report shows for all data sources on the network. The default is an overview report which is an aggregated report rolling up all reported interfaces for all collecting devices.

Subnet Reporting

Application Auditor also provides overview reporting based upon monitored subnets and Individual IP addresses configured within your installation. The following subnet reports are available from accessing the ‘subnets’ menu item in the navigator menu:

Subnet Overview

Subnet overviews provide a summarised view over time of the key areas based at both subnet and individual IP levels. These reports show top talkers, domain and Geo-IP information as well as subnet to subnet statistics to help identify root cause of network utilisation issues.
