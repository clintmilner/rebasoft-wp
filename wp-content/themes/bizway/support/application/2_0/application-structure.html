Application Structure

Application Auditor has been designed with ease of use in mind.

The screen shot below shows the main areas for interacting with Application Auditor:



  1. The Navigator allows individual sources or data types to be selected. Subnet reporting is also selectable from the Navigator.

    note: When using the Navigator to select items, the reporting period is reset to the default defined in System settings


  2. The Admin selector is only shown to "admin" class users and provides the navigation to the Admin & configuration functions in Application Auditor.
  3. Reports and forensic selections are displayed in the Display frame.
  4. The Toolbar is designed to be similar to Auditor Core. It implements an updated time-line manipulation function to enable users to more accurately select reporting spans. Similarly, like Auditor Core, the toolbar allows users to select the output format for reporting. The default is HTML, which displays on the screen, with PDF, XML, JSON and CSV options also available as required.

    Key

    		Allows users to go back to the previous screen or report
    		Reloads the display frame
    		Prints the display frame
    		Access on-line help at www.rebasoft.net/support

    A date picker allows users to set a start and end range by date, month and year. Time is selected using the hour and minute slider

    Once report, format and dates have been selected you must press the OK button to execute the report:
