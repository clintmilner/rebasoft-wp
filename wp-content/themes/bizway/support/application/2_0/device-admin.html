Device Settings

Once a device has been configured to export flow information to Application Auditor, it is automatically registered by the system. Device settings allows administrators to fine-tune and configure settings on a per device basis as needed. To access device settings, select 'Devices' from the Admin drop down

In Device settings, Application Auditor displays the list of automatically discovered devices together with other pertinent information. Application Auditor shows this summary in a table showing the type of data being received (and the number of packets received) together with SNMP polling statistics (if any):

note: A device can export more than one source of application flow data (for example a Juniper Networks SRX can export jFLow, AppSecure and Firewall session logs - this single devoice would count as 3 sources for licensing purposes)

 

To edit a device, simply select from the list. Administrators can configure the following elements:

  • Override the Display Name
  • Set SNMP Version (1, 2 or 3)
  • Device specific SNMP Community
  • Sampling rate override (if required)
