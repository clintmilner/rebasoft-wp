Installing Application Auditor

Installation is achieved by executing the Application Auditor installation wizard. This wizard guides administrators through the installation or updating process and allows the specification of a number of parameters – it should only take a few minutes to install the system.

The installer can be run whether performing a new installation or updating an existing system. In the case of an upgrade, data is preserved; Rebasoft does, however, advise administrators to perform a data back-up regularly using pgAdmin (see appendix) and especially prior to system upgrade.

note: Due to windows security, we recommend that the installer be run as the Local Administrator

Logging in for the first time

Users and administrators can access Rebasoft Application Auditor from a workstation with a compliant browser by opening the following address:

http://<IP or DNS address of server>:<port>
