Searches

Rebasoft Auditor’s powerful search engine allows users to search for items in the database. Most users will use the search as way to quickly navigate to the point of interest. Searching can match on full or partial strings entered in the search box

Search accuracy can be improved by specifying a more complete string. For example, searching on “10.100.1” will bring up all the matching records containing this string – in this case matching IP addresses and devices.

Searches return results for MAC, User ID, IP address, Hostnames, Device details, Interfaces and VLANS:


Tip:

If you click the search button without entering any parameters, Rebasoft Auditor returns a search of all entries in the database.


Only when you select a search result does Rebasoft Auditor query the database (default 8 days) to see when the item occurred. Information displayed in timelines allows users to view longer or short periods of time as required. Using this method Auditor, for example, does require users to know when someone was on the network to see if they've ever logged in.
