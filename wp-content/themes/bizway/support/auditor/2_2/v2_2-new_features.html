New features for V2.2

Rebasoft is continually refining and developing Auditor, v2.2 sees the introduction of new features based upon customer requests and requirements:

  • Hostname/DNS forensic reporting
  • Support for CDP & LLDP for network device discovery
  • Support for CDP & LLDP for hostname discovery (useful for VoIP network and IP Phones)
  • Additional device support for Checkpoint, Alcatel, Juniper and Cisco 6500 FWSM‟s
  • Ability to integrate directly with Visual Network Systems NetFlow Tracker
  • Ability to build subnet filters to selectively work with integrated systems
  • Ability to bulk import tags for interfaces
  • Ability to deploy Auditor database on a separate server
  • Ability to specify how long data is kept for in the database in a very granular fashion
  • Significant reporting improvements and performance improvements in data storage
  • REST API for programmatic access to Auditor data
  • Active test now

To support these new features, there have been a number of minor changes to the application which are described in the sections below. A list of changes can be viewed in Rebasoft Auditor - V2.2 Release notes
