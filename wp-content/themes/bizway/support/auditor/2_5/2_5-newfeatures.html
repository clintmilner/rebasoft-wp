New features for V2.5

Rebasoft is continually refining and developing Auditor Core. V2.5 is a maintenance release that adds new features requested by clients:

  • SNMP V3
  • Support for Juniper Wireless LAN controllers (formerly Trapeze)
    This new support paves the way for additional WLAN support for other vendors

Rebasoft recommends that clients upgrade to this new release if any of the features are needed.

note:

Upgrading from V2.4 to V2.5 does not require a new license key
