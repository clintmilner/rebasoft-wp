New features for V2.6

Rebasoft is continually refining and developing Auditor. V2.6 introduces new features built upon the previous releases and based upon customer requests and requirements:

  • Security module with focus on supporting IT Risk Management:
    • Network Edge discovery and approval processes
    • Alerting enhanced to record when a user access port changes to an network uplink port – identifying potential rogue access points/switches
    • End device coverage and approval processes including passive fingerprinting

Enhancements to Auditor:

  • Event management improved to enable handling of large volumes of alerts

Rebasoft recommends that clients upgrade to this new release if any of the features are needed.

note: Upgrading from V2.5 to V2.6 does not require a new license key unless licensing the security feature set
