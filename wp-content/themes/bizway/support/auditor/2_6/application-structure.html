Application structure

Auditor has been designed with ease of use in mind. From V2.4 onwards views are selectable. Depending on the privileges that a user has been set (and the license type purchased), the view selector may have one or more options.

The screen shot below shows the main areas for interacting with Auditor:

 

  1. The Search input allows searching on any string. See Searches for more information on searching.
  2. The View selector shows the available views options. In 2.4 (or later) this allows users to view Capacity or Operations reporting. Admin users will see all licensed views, non-admin users will see those views they are authorised to see.

    note:

    Clicking the button shows the home page within the selected view.


  3. The Admin selector is only shown to "admin" class users and provides the navigation to the Admin & configuration functions in Auditor.
  4. The Navigator lists the browse structure depending on the view and Tags implemented.

    note:

    When using the Navigator to select items, the reporting period in the display frame is reset to the default period of 24 hours


  5. Search results or Navigator selections are displayed in the Display frame.
  6. Finally, the Event viewer displays system information, including license status and utilisation and a link to system event alerts.

Navigating reports

From each of the ribbon graphical reports, it is possible to "zoom" the time frame in and out by selecting in the graph. Simply left click at the start & drag the mouse to the desired end time frame. The menu will automatically appear showing the options available.

note:

When zooming on Dashboard, Site or Device overview on one selected report, Auditor will automatically zoom the other visible reports to the same timeframe

In addition to zooming, it is possible to select reports for the selected site and timeframe.

Timeline manipulation

In the forensics reports, Auditor displays timelines for easy visual assimilation of information. It is possible from these time lines to move back and forward along the time line at a particular resolution, and to zoom in or out.

The default display for all time-lines is 8 days

It is also possible to view the detailed data behind the time line by clicking "Details" underneath the timeline.

To aid visualising information on screen, it is also possible to arrange timelines and details to suit user requirements in lining up IP and MAC over time.
