Events

Auditor can generate events and log these to the in-built event viewer for further action. Optionally these events can be sent to one or more external syslog servers.

Events are classified by type and filterable by using the hyperlinks in the menu below:

Key event types include:

Event Type

Trigger

System

System status messages

Watchlist

Watchlist hits

Whitelist

Whitelist hits

Device

Device add/snmp status messages

User

User login messages

Interface

Interface status change

Network Edge

New network edge discovered or status/type changed

MAC Coverage

Records a change to the status or type classification by a user

 

Time frame can be filtered by click-dragging on the event chart for the selected time frame:

Similarly, the start and end times can be selected using the time navigator on the toolbar:

Clicking any of the hyperlinks will filter the event list to display the relevant information.

 

Events can be deleted, acknowledged or un-acknowledged. These actions do not affect any events that have already been forwarded to any external syslog received configured in System settings administration.
